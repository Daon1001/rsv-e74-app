# E-7-4 비자 컨설팅 앱 (RSV)

E-9 → E-7-4 (숙련기능인력) 비자 전환을 위한 **사전 진단 + AI 보강 플랜** 도구.

## 주요 기능

1. **결격사유 자가진단** — 형사범/세금체납/출입국위반/불법체류 등 신청 불가 사유 자동 체크
2. **점수제 시뮬레이션** — 법무부 K-point E74 기준 300점 만점 자동 계산
   - 평균소득 (120점) · 한국어 (120점) · 나이 (60점) · 경력 (50점) · 학력 (35점) · 가점
   - 합격선 200점 + 평균소득/한국어 각 최소 50점 필수 체크
3. **Claude AI 맞춤 플랜** — 부족한 점수를 채우는 최단 경로 6개월 로드맵 자동 생성
4. **사용자 승인 시스템** — 관리자 승인 후 이용
5. **사용량 추적** — 사용자별 토큰/비용 (USD/KRW) 실시간 집계

## 비자 정책 참고 (2025~2026 기준)

- **신청 자격**: E-9/E-10/H-2로 최근 10년간 4년 이상 체류 (사회통합 3단계+ 이수시 4년 충족 인정)
- **연봉**: 2,600만원 이상 (농축산·어업은 2,500만원)
- **현 사업장**: 1년 이상 근무 + 추천서
- **합격**: 300점 만점 중 200점 이상 + 평균소득/한국어 각 50점 이상
- **한국어 요건 유예**: 2026.12.31까지 한시 (이후 TOPIK 2급+ 필수)
- **신청**: 2025년부터 상시 신청 (분기별 모집 폐지), 하이코리아 온라인
- **2025 쿼터**: 35,000명 (확대)
